# 🚁 Aerial Object Classification & Detection

## Project Overview
A deep learning project to classify aerial images as **Bird** or **Drone** using CNN and Transfer Learning.

## 🎯 Results
| Model | Accuracy |
|-------|----------|
| Custom CNN | 86.51% |
| Transfer Learning (MobileNetV2) | 95.81% |
| YOLOv8 Detection (mAP50) | 68.5% |

## 🛠️ Technologies Used
- Python
- TensorFlow & Keras
- MobileNetV2 (Transfer Learning)
- YOLOv8 (Object Detection)
- Streamlit (Deployment)
- Google Colab (Training)

## 📁 Dataset
- Train: 2662 images (Bird & Drone)
- Valid: 442 images
- Test: 215 images

## 🚀 How to Run
1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
